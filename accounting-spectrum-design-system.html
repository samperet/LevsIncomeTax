Design System / Accounting Firm
Spectrum-inspired fintech system

Trustworthy accounting interfaces for clear financial decisions.

A restrained enterprise design system for accounting firms, built around visible hierarchy, reliable color semantics, dense but readable dashboards, and low-friction workflows for partners, staff accountants, and clients.

View dashboard pattern Review tokens
Trust before flair Neutral surfaces, stable layout, and measured accents communicate reliability without visual noise.
Clarity under load Tabular numerals, aligned columns, and scannable groupings make variance and risk easy to parse.
Accessible by default Contrast, focus, touch targets, and semantic structure are built into the system primitives.
Action proximity Actions appear beside the financial object they affect, reducing memory burden and mistakes.

Color Tokens

Muted enterprise fintech colors separate brand, status, and financial meaning. Blue carries product trust, teal marks verification, amber flags review, and red is reserved for destructive or overdue states.

Density Rules

FormsUse 16px gaps within a form section and 24px between logical groups.

DashboardsKeep KPIs above tables and pair each metric with a short operational note.

System Rules

Use these constraints to keep product decisions coherent as the accounting platform grows.

Implementation guidance
Use blue for actions onlyDo not use brand blue as a decorative fill when it competes with workflow actions.
Expose source and basisEvery dashboard should state reporting period, accounting basis, and currency near the title.
Align money rightCurrency, percentages, and hours use tabular numerals and right alignment in comparison tables.
Keep warnings specificAlerts must include the account, variance, or compliance item that needs attention.
Prefer progressive disclosureShow summary, risk, and next action first; move deep ledger detail behind a focused view.
Design for repeated workReduce ornamental layout shifts and preserve familiar control placement across client contexts.
